# E-Shopping 

- Developed with Flutter & Dart
- Back-end as Firebase

## Features
- Adding Product to Cart
- Adding Favorite Products
- Ordering a Product


[Flutter](https://flutter.dev/docs/cookbook)
[Firebase](https://firebase.google.com)
